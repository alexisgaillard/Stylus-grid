STYLUS SEMANTIC GRID

Fork on

A simple and semantic grid system written in Stylus, this mixin give you freedom to create completely custom grids, in less than a line, ~ 2k minified.

Get started:

01 Clone

          
# git clone git://github.com/alexisgaillard/stylus-grid.git
# cd stylus-grid
# stylus stylus-grid.styl -o path/to/your/stylesheets

02 Import and init

          
@import "stylus-grid"
grid()

Your marColp should look like this:

          
<div class="Grid">
  <div class="Col__1">...</div>
  <div class="Col__11">...</div>
  <div class="Col__2">...</div>
  <div class="Col__10">...</div>
  ...
</div>

All settings list:

          
 container-selector-name = Grid
 column-selector-name = Col  //will be suffixed with __{column}
 container-max-width = 1272px  //value in px or false
 column = 12  // minimum > 0
 gutter-w = 1%  //percentage can be set to 0
 gutter-inner = false  //if true set de gutters as padding instead of margin
 pre-post = true  //if false avoid the output of the .Pre and .Post classes
 deepness = 6  //define the number of decimals on the percentage's width values
